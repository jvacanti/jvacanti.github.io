Say it with Latitude

We all know the weather is warmer near the equator, but how would you prove it?

In this project I analyze how weather changes in relation to the equator. I used the OpenWeatherMap API to create a dataset of more than 500 cities across the globe. Then using Matplotlib, I created plots of weather vs. latitude including temperature, cloudiness, wind speed and humidity.
